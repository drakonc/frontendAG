## Descripción

**FrontEnd** Echa en **Angular** para el inventario de los equipos y control de usuarios del area de Infraestructura.

## Instalación

```bash
$ npm install
```

## Correr la aplicación

```bash
# Modo Desarrollo
$ npm run start

# Transpilar
$ npm run build
```
